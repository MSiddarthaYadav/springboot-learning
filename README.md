## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/MSiddarthaYadav/springboot-learning.git
```

2. Open the project in VS Code or IntelliJ IDEA.

3. Configure the database in `application.properties`.

4. Build the project

```bash
mvn clean install
```

5. Run the application

```bash
mvn spring-boot:run
```

6. Test the REST APIs using Postman.
